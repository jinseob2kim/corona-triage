## :pushpin: 소개

- Covid-19 확진자를 중증도에 따라 분류, 의료자원을 고려한 치료 전략에 도움을 준다.
- Corona-triage는 경기도 코로나19 긴급대책단의 의뢰로 진행하는 프로젝트입니다. 

## :memo: 중증도 분류 기준
